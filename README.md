# Classification

Preprocessing data, analysis and find insights.

## Overview
This repository contains code for data preprocessing, exploratory data analysis, and generating actionable insights from breast cancer datasets using classification models.

## Structure
- `data/` - Raw and processed datasets
- `notebooks/` - Jupyter notebooks for analysis
- `src/` - Source code for preprocessing, analysis and building classification models.
- `results/` - Identifing the best Classification models of breast cancer dataset with good accuracy.

## Main Branch
This is the main development branch for the Classification project.
